# Typescript 

TypeScript is an open-source programming language that will work on any browser, OS, and environment that supports JavaScript. It adds *types* to JavaScript, which are designed to help speed up the development process through catching errors before running the JavaScript code. I'll use a [TypeScript tutorial](https://www.typescripttutorial.net/) as a guide for working through the details of this programming language. 

To get started, let's use our Jupyter environment to keep things tidy and to the point. Install tslab globally and then the kernal with tslab:

```console
uname@os:~$ npm install -g tslab
uname@os:~$ tslab install
```
