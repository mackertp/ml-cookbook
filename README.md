# ml-cookbook

This repository serves as a reference to tools and configurations that help with creating machine learning based applications. 
It focuses on Aurélien Géron's [Hands-On Machine Learning with Scikit-Learn & Tensorflow](https://www.google.com/books/edition/Hands_On_Machine_Learning_with_Scikit_Le/bRpYDgAAQBAJ?hl=en&gbpv=1&printsec=frontcover)

In the *languages* folder, there are configs that will help you with using and understanding different programming languages. Not every developer is always
going to want to use python for a project, so it is good to have an idea of how other languages can be used.

To get started and follow along, the code for hands on ml will use a [Portry](https://python-poetry.org/) environment. Configuration for other languages
will be detailed in the respective folders.

```console
uname@os:~$ poetry install
uname@os:~$ poetry shell
uname@os:~$ jupyter-lab
```
Happy hacking!
