# Hands-On Machine Learning

This folder follows Aurélien Géron's [Hands-On Machine Learning with Scikit-Learn & Tensorflow](https://www.google.com/books/edition/Hands_On_Machine_Learning_with_Scikit_Le/bRpYDgAAQBAJ?hl=en&gbpv=1&printsec=frontcover). Aurélien is a former Googler and was the founder and CTO of a couple of french companies, Wifirst and Polyconseil. He's also worked as an engineer in finance, defense, and healthcare, and also brings several academic accolades.

I think this book is a great starting point for learning about data science and machine learning. The objectives for the reader are to:

- Explore the machine learning landscape, particularly neural nets
- Use scikit-learn to track an example machine-learning project end-to-end
- Explore several training models, including support vector machines, decision trees, random forests, and ensemble methods
- Use the TensorFlow library to build and train neural nets
- Dive into neural net architectures, including convolutional nets, recurrent nets, and deep reinforcement learning
- Learn techniques for training and scaling deep neural nets
- Apply practical code examples without acquiring excessive machine learning theory or algorithm details

The poetry environment from the parent folder should contain all of the dependencies you need to follow along with my notebooks on the book. Please make sure you recognize [Aurélien](https://github.com/ageron), he has made his source code for the book
publicly available [here](https://github.com/ageron/handson-ml3). My notebooks that follow the book have been excluded from the repository, please reference Aurélien's code and purchase his book!
